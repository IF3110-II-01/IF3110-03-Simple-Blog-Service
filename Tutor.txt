Install Maven	
Fork lalu clone https://github.com/chamerling/heroku-cxf-jaxws/
	
	Masuk ke direktori tempat clone di atas
    Login to Heroku : 'heroku auth:login'
    Create the app on heroku : 'heroku create -s cedar'
    Push the app to heroku : 'git push heroku master'
    Open the new app : 'heroku open'

Buat file .bat di root SimpleBlog (Jalankan setiap ingin disinkronkan dengan server, saat menambah fungsi, dll.)
wsimport -keep -d nbproject -s src/java <isi alamat yang ada di heroku open>

Pada: D:\Github\heroku-cxf-jaxws\src\main\java\org\chamerling\heroku\service
Di file HelloService.java buat berbagai header fungsi dan parameter
Di file HelloServiceImpl.java buat implementasinya
